Melalui website ini pengguna dapat mengetahui informasi terkait Covid-19 seperti data penyebaran covid-19 secara terbaru dan detail, jumlah kasus covid-19 baik yang meningkat maupun menurun, informasi harian yang terupdate, serta pengguna dapat mengetahui resiko-resiko dari virus covid-19. Website ini menggunakan teknologi sebagai berikut :
  • Bootstrap 4 untuk mempercantik tampilan
  • Senna.js framework frontend javascript untuk mempercepat loading halaman
  • API dari https://covid19.go.id/ untuk mendapatkan data covid-19
  • Jquery untuk mempermudah pemanggilan API
  • Fontawesome untuk ikon
